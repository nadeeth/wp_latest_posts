== Installation ==

1. Upload the 'wp_latest_posts' folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

OR

1. Compress 'wp_latest_posts' folder as a zip file. (wp_latest_posts.zip)
2. Login as wp admin.
3. Go to "Plugins --> Add New"
4. Click Upload
5. Brows and select the file(wp_latest_posts.zip), click "Install Now" button.
6. Activate the plugin
